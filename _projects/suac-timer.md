A browser-based archery timer developed for **Swansea University Archery Club** to support training sessions and practice rounds.

This project adapts an original timer created by **Mike Brockington**, with additional features and improvements designed to better suit club shooting sessions.

## Live Example

You can try the timer here:

<https://arthurb-dev.github.io/SUAC-Timer/>

## Source Code

The project is open source and available on GitHub:

<https://github.com/ArthurB-Dev/SUAC-Timer>

## Background

This archery timer was developed as a simple and accessible timing tool for Swansea University Archery Club. While designed with our club sessions in mind, it can be used by **any archery club, coach, or individual archer** who needs a straightforward timing system for practice or training.

The tool runs entirely in a **web browser**, meaning it requires no installation and can be used on laptops, tablets, or phones at the range.

The project is **actively being developed**, with additional features and improvements planned for future updates.

![Desktop View](assets/img/project-images/timer/Screenshot 2026-03-08 183627.png){: width="972" height="589" }


## Features

- Customisable shooting timer
- Simple and intuitive interface
- Audio alerts for timing cues
- Real-time display of remaining time
- Fast call and equipment failure support
- Timing statistics and reporting
- Browser-based (no installation required)
- Free and open source

## Development

This project is still under active development. Planned improvements include:

- Additional timing modes
- Improved mobile layout
- Expanded configuration options
- More detailed timing analytics

## Usage

1. Open the timer in your web browser.
2. Configure the timer settings for your shooting format.
3. Start the timer to run a shooting end.

Because the timer runs entirely in the browser, it can also be downloaded and run locally if needed.
